## How to Play

1. **Open the game**: Simply open `index.html` in your web browser
2. **Choose mode**: Select Normal Mode or Ninja Mode
3. **Select play type**: Choose to play with a friend or against a bot
4. **Enter names**: Input player names (or use defaults)
5. **Choose symbol**: Select X or O (X always goes first)
6. **Play**: Click on cells to place your symbol
7. **Win/Lose**: Follow the rules based on your selected mode

## Game Rules

### Normal Mode
- Players take turns placing X or O
- First player to get 3 in a row (horizontal, vertical, or diagonal) wins
- If the board fills up with no winner, it's a draw

### Ninja Mode
- **You LOSE** if you make a line of 3
- **You WIN** if your opponent is forced to make a line of 3
- Draw occurs if the board is full and no one has formed a line of 3
